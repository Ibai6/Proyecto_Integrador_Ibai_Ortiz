# Aventura del Héroe

Proyecto realizado en **HTML, CSS y JavaScript**.

El juego consiste en guiar a un héroe por un tablero 10x10 hasta llegar a un cofre, utilizando tiradas de dado que determinan los posibles movimientos.

---

## Descripción del juego

El jugador introduce su nombre y comienza la partida.  
En cada turno se tira un dado que genera un número aleatorio entre 1 y 6.  
Según ese número, se muestran las casillas a las que el héroe puede moverse.  
El objetivo es llegar al cofre con el menor número de tiradas posible.

El juego guarda el mejor resultado utilizando `localStorage`.

---

## Cómo ejecutar el proyecto

1. Descargar o clonar el repositorio.
2. Abrir el archivo `index.html` en un navegador web.
3. No se requiere instalación adicional.

---

## Cómo jugar

1. Introducir un nombre válido y pulsar **Aceptar**.
2. Pulsar **Jugar** para iniciar la partida.
3. Pulsar **Tirar dado**.
4. Hacer clic en una de las casillas marcadas en rojo.
5. Repetir hasta llegar al cofre.
6. Al finalizar, se muestran las tiradas y el récord.
7. Pulsar **Siguiente partida** para volver a jugar.
